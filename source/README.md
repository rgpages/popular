## Command line

```bash
# First:
$ cd popular/source/

# Next, install:
$ npm install
# ... or update Grunt dependencies:
$ npm update

# Update Bower packages:
$ grunt plugins
# ... or:
$ npm run plugins

# Watch:
$ grunt watch
# ... or:
$ npm run watch

# Development build:
$ grunt
# ... or:
$ grunt dev
# ... or:
$ npm run dev

# Production build:
$ grunt prod
# ... or:
$ npm run prod
```

**Note:** If [Grunt](http://gruntjs.com/) isnâ€™t installed globally, then roll with the `$ npm ...` commands.
